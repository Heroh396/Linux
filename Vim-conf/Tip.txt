- Copy text in vim:
        + Command:      :set paste
        + i
        + Paste:        Ctrl + Shift + V

- Copy text in other file:
        + Comand mode
        + :r file_path

- Using command in vim:
        + :!<command>

- Top, down: gg, G

- Copy line:
	+ y <number> y
	+ Paste: p (head line), P (tail line)

- Delete:
	+ dd (all line), D (from pointer to end line)

- Change ESC: Shift + {

- Delete char: x

- Mode: :q, :q!, :w, :w!, :wq
